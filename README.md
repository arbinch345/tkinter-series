# 📖 What is Tkinter?

**Tkinter** is Python's **standard Graphical User Interface (GUI) library** that allows developers to create desktop applications with windows, buttons, menus, text boxes, dialogs, images, and many other interactive components.

Tkinter acts as a bridge between Python and the **Tk GUI Toolkit**, providing a simple yet powerful way to build cross-platform desktop applications that run on **Windows, macOS, and Linux**.

Unlike web development, where applications run inside a browser, Tkinter applications run directly on your operating system as native desktop programs.

# ⚙️ How Tkinter Works

Tkinter follows an **event-driven programming model**.

Instead of executing code from top to bottom continuously, your application waits for user actions such as:

- Clicking a button
- Pressing a key
- Selecting a menu
- Moving the mouse
- Typing text

When an event occurs, Tkinter calls the function associated with that event.
